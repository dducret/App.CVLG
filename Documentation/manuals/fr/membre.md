# Manuel utilisateur membre

## Public

Ce guide s'adresse aux membres standards qui utilisent l'espace membre de CVLG.

## Navigation principale

- `Mon profil` : mettre a jour les informations personnelles, la langue, l'adresse et le mot de passe.
- `Mes cotisations` : consulter les cotisations annuelles et marquer une cotisation comme payee dans la version actuelle.
- `Mes tickets` : acheter des packs de tickets et consulter les achats.
- `Reservations` : reserver ou annuler une place de navette a venir.
- `Mon historique` : consulter les reservations a venir et passees.

## Connexion

1. Ouvrir la page de connexion.
2. Selectionner la langue si necessaire.
3. Se connecter avec l'email ou l'identifiant et le mot de passe.
4. Apres connexion, un membre est redirige vers `Reservations`.

## Mon profil

Utiliser `Mon profil` pour maintenir :

- Prenom, nom, surnom, numero de mobile
- Date de naissance, sexe, nationalite
- Rue, numero, code postal, ville, pays
- Langue preferee
- Mot de passe

La page affiche aussi :

- L'email de connexion
- Le code de role
- Le type de membre
- Les licences et leurs dates de validite

## Mes cotisations

Utiliser `Mes cotisations` pour :

- Consulter les cotisations annuelles
- Voir le montant, le statut, le moyen de paiement et la date
- Utiliser l'action `Payer` sur une cotisation non reglee

Comportement actuel :

- Le bouton `Payer` marque la cotisation comme payee avec un moyen de paiement simule.

## Mes tickets

Utiliser `Mes tickets` pour :

- Voir le prix unitaire du ticket
- Voir le nombre de montees restantes
- Acheter un nouveau pack de tickets par quantite
- Consulter l'historique des achats

Important :

- Les reservations dependent du nombre de montees restantes.

## Reservations

La page `Reservations` est l'ecran principal d'utilisation pour un membre.

Vous pouvez :

- Voir les navettes disponibles pour aujourd'hui et les deux jours suivants
- Ouvrir la confirmation de reservation en cliquant sur un horaire
- Ajouter un nom d'invite pour un passager biplace
- Consulter les reservations en cours
- Annuler une reservation a venir
- Lire les regles de reservation actives affichees a l'ecran

La page affiche aussi :

- Le nombre de places restantes par navette
- Le chauffeur et le vehicule lorsqu'ils sont affectes
- Votre propre statut : reserve ou liste d'attente

## Regles de reservation

La version actuelle peut appliquer les regles suivantes :

- Un membre ne peut pas reserver deux fois la meme navette.
- Un membre peut etre bloque s'il tente de reserver deux navettes a la meme date et a la meme heure.
- Le nombre de reservations confirmees peut etre limite par jour.
- Une demande supplementaire le meme jour peut passer en liste d'attente au lieu d'etre confirmee.
- La taille de la liste d'attente peut etre limitee par navette.
- Le nombre de reservations en attente peut aussi etre limite par membre et par jour.

## Quand Une Reservation Est Refusee

La reservation est refusee dans les cas suivants :

- Votre compte membre n'est pas autorise a reserver.
- Vous n'avez plus de montee disponible.
- La navette n'est plus ouverte a la reservation.
- Vous avez deja reserve cette navette.
- Une regle bloque la demande.

## Mon historique

Utiliser `Mon historique` pour :

- Consulter les reservations a venir avec leur statut et le QR code
- Consulter les reservations passees recentes

## Notes pratiques

- Si votre situation membre n'est pas reguliere, la page de reservation affiche un avertissement et bloque la reservation.
- Si vous n'avez plus de montees, achetez des tickets avant une nouvelle reservation.
- Un changement de langue dans `Mon profil` s'applique aux ecrans suivants apres enregistrement.
